Key Findings

Insights from analyzing industry adoption patterns

01

Professional Services Dominates

NAICS sector 54 (Professional, Scientific, and Technical Services) leads adoption across all three agents, representing 22-28% of repositories. This is expected as software development is a core activity in this sector.

02

Information Sector Strong Second

NAICS sector 51 (Information) consistently ranks second across all agents, representing tech companies, publishers, and data processing organizations heavily invested in software development.

03

Claude Code Shows Fastest Growth

Claude Code demonstrates the steepest adoption curve, particularly in late 2025, with January 2026 alone seeing 111,000+ new repositories‚Äîmore than double Copilot's and Codex's monthly growth.

04

Finance and Healthcare Accelerating

Sectors 52 (Finance) and 62 (Healthcare) show accelerating adoption rates, suggesting growing trust in AI coding tools for regulated industries.
